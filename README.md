# E-Commerce Application

Full-stack E-Commerce platform built using:

## Backend
- Spring Boot
- Spring Security
- JWT Authentication
- Hibernate / JPA
- MySQL

## Frontend
- React
- React Router
- Axios
- Bootstrap

## Features

### User
- Register/Login
- Browse Products
- Add to Cart
- Place Orders
- View My Orders

### Admin
- Add Product
- Update Product
- Delete Product
- Admin Dashboard

## Tech Stack
React + Spring Boot + MySQL + JWT
