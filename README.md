# E-Commerce Application

A full-stack E-Commerce web application built using Spring Boot, React, MySQL, and JWT Authentication.

## Features

### User
- Register & Login
- Browse Products
- Search Products
- Add Products to Cart
- Place Orders
- View My Orders

### Admin
- Add Product
- Update Product
- Delete Product
- Admin Dashboard

## Backend
- Spring Boot
- Spring Security
- JWT Authentication
- Hibernate / JPA
- MySQL

## Frontend
- React
- React Router
- Axios
- Bootstrap

## Testing
- JUnit 5
- Mockito

## Tech Stack
- React
- Spring Boot
- Spring Security
- JWT
- Hibernate
- JPA
- MySQL
- Maven
- JUnit
- Mockito

## Run Backend

```bash
mvn spring-boot:run
```

## Run Frontend

```bash
npm install
npm run dev
```
